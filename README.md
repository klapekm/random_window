# Java Random Data Generator
Generates either text or binary files composing of random numbers generated using users choice of java generator.
## Window Version
To use run the JDRGuiRun file, then follow those steps:

1. Select the date type(bin/txt),
2. Input the amout of data(MB or num amount depending on your previous step),
3. Select the generator,
4. Input the file path or click the generate button once, wich will create an example path for you,
5. Click generate button,
6. Click the clear butotn to clear the file path field,
7. Repeat.

Immediate help with any problem is avalible under the help button in the toolbox.
## Console Version
Locate the JRDCli file and then:
1. Run the file, 
2. Input the data type(bin/txt),
3. Write the name of the generator you wish to use or type 'h' to get a list of all the avalible ones. You can also write the index of the generator in the list.
4. Write the file path or write h to create one(this will imediatly generate the file under the genearated path, no editing is possible),
5. Repeat.
